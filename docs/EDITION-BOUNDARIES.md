# 蓝色版与橙色版边界

## 蓝色办公卜卜

- 面板版本标识：`blue-bubu`
- 宠物 ID：`bubu-office`
- 安装包只允许包含 `pet/bubu-office`
- 不包含荧光棒、橙色沙滩椅素材、橙色皮肤切换或橙色尺寸表
- macOS 继续使用 `io.github.mayday-materials.bubu-quota-panel`
- Windows 继续使用 `%LOCALAPPDATA%\BubuPet`

## 橙色卜卜

橙色卜卜是独立项目。今后的运行程序、启动项、安装目录、资源目录、
健康检查文件和发布包都不得复用蓝色版标识。橙色资源不得进入蓝色安装包。

两个角色可以同时存在于仓库的设计预览中，但发布构建必须按上述边界隔离。
