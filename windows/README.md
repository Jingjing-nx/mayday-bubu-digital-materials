# 卜卜 Windows 版

## 安装

1. 完整解压 `Mayday-Bubu-Windows-10-11-Codex-Only-v1.0.3.zip`。
2. 双击 `安装卜卜-Windows.cmd`。
3. 安装器会关闭正在运行的 Codex/ChatGPT；安装完成后重新打开即可。

安装器会先退出正在运行的客户端，以临时目录校验并完整替换宠物，再安装 WPF 额度面板。面板失败不会影响宠物本体。

- 完整版 v1.0.1：保持不变，显示 Codex 额度与 BTC/ETH 行情。
- `Codex-Only` v1.0.3：只显示 Codex 额度，面板自动缩短，并且不会请求 BTC/ETH 行情。

`v1.0.3` 会先清理旧宠物目录、校验新图集并重新写入宠物选择；同时修复多显示器 DPI、宠物缩放和低帧率环境中的面板分离。

## 要求

- Windows 10/11
- x64 或 ARM64
- 推荐启用系统自带 Windows PowerShell 5.1、WPF 和 WSH
- 不需要管理员权限、Python、Node.js、.NET SDK 或 API Key

## 低延迟跟随

- 优先跟随 Codex 原生宠物窗口，统一使用每显示器 DPI v2 的物理坐标计算位置。
- 排除输入法候选框和其他窄长弹窗，输入文字时面板不会跟随候选框移动。
- 使用系统级不激活窗口样式，面板不会抢走输入焦点。
- 使用 Windows 桌面合成器逐帧事件移动面板；软件渲染、远程桌面和低刷新率环境会由 33 ms 定时器兜底。
- 拖动或缩放宠物时持续刷新锚点，只暂停额度、健康日志以及完整版的 BTC/ETH 刷新，停止后自动恢复。
- 原生窗口不可用时才使用保存坐标回退。

## 兼容入口

- `兼容安装-只装宠物.cmd`：公司电脑限制 PowerShell 时使用。
- `修复或启动看板.cmd`：覆盖并重启额度面板。
- `检查安装环境.cmd`：在桌面生成脱敏报告。
- `卸载卜卜-Windows.cmd`：卸载宠物和面板。

## 数据

Codex 额度只从本机服务读取；完整版的 BTC/USDT 和 ETH/USDT 来自 Binance 公共现货行情，`Codex-Only` 版不会连接行情接口。诊断日志会隐藏用户路径、邮箱和常见 Token，并且不记录真实额度百分比。
