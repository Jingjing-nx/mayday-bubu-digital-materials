# 卜卜 macOS 版

## 安装

1. 按需要完整解压 `Mayday-Bubu-macOS-Universal-16.zip` 或 `Mayday-Bubu-macOS-Universal-Codex-Only-16.zip`。
2. 双击 `安装卜卜-macOS.command`。
3. 如果出现“Apple 无法验证”提示，点“完成”，不要点“移到废纸篓”。
4. 双击 `安装被拦截-打开隐私与安全.html`；页面会尝试自动跳转，如果没跳转就点蓝色按钮。
5. 在“隐私与安全”中点击“仍要打开”或“Open Anyway”，输入 Mac 登录密码确认，再重新双击安装文件。
6. 如果跳转仍有问题，双击 `如果仍无法打开-Apple官方步骤.webloc`。
7. 重新打开 Codex；安装器会优先自动选中卜卜。

“仍要打开”按钮只会在尝试启动安装文件后出现，并会保留约一小时。
[查看 Apple 官方的允许步骤](https://support.apple.com/guide/mac-help/mh40616/mac)。

## 要求

- macOS 12.3+
- Apple 芯片或 Intel Mac
- 已安装并登录 Codex

## 功能

- 安装蓝色 Codex v2 宠物卜卜，并自动设为当前宠物。
- 原生 AppKit 额度面板，约 30 ms 跟随卜卜；面板整体随卜卜同比放大缩小，箭头和卜卜可见中心对齐，箭头尖端到头顶固定为 14 个逻辑像素。
- 实时测量卜卜的实际可见图像；即使透明外层窗口尺寸没变，面板外框、文字、五个球、进度条、任务行、按钮、圆角和箭头也会使用同一个比例缩放。兼容 Retina、非整数缩放、多屏负坐标和旧版 Codex 的 `anchor` 记录。
- 找不到宠物窗口时会读取 Codex 保存的位置；短暂切屏会保留最近一次精确位置，超过 0.5 秒仍无法定位则隐藏面板，避免面板与宠物长期分离。
- 退出 Codex 后面板会自动消失；点击面板上的“隐藏”会完全收起，不再残留桌面小方块。双击卜卜即可显示，再双击即可隐藏，之后可一直循环切换；菜单栏“卜卜”仍作为备用恢复入口。
- 安装结束前会检查面板进程和运行状态文件，首次启动失败会自动重试。
- Codex 额度每 5 分钟更新。
- 任务进度约每 2 秒更新，会用任务唯一 ID 对应 Codex 左侧任务列表中的正式名称。四种状态使用同一套角色图标：正在执行为蓝色眼镜球与持续旋转的蓝色循环箭头，等你确认为橙色手机球与黄色问号，已完成为粉色球与绿色勾，执行失败为红色球与红色叉。Codex 自动化生成的定时任务不会计入面板。带蓝点、尚未查看的已完成或失败手动任务会保留在面板底部，用户在 Codex 中点开任务、蓝点消失后，面板同步移除该行。多个手动任务按开始时间依次向下排列，最多显示 5 行。任务名称与已读状态只在本机读取和显示，不写入日志，也不上传；没有相关索引时才回退到任务消息摘要与短时完成提示。
- 该 `Codex-Only` 版只保留额度，面板会自动缩短，并且不会请求 BTC 行情。
- 不需要管理员权限或 API Key。

## 检查与卸载

- `检查卜卜-macOS.command`：检查宠物、Universal 2 架构、签名、面板进程、运行状态、跟随定位和额度；完整版本还会检查行情。
- `卸载卜卜-macOS.command`：只移除卜卜和本项目面板。
